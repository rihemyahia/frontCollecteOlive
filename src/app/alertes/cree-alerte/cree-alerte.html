⚠️ Comment est calculée l'urgence ?

Le système évalue automatiquement le niveau d'urgence de votre alerte en fonction de plusieurs facteurs. Voici comment cela fonctionne :

🔴

Type d'alerte

Certains types d'alertes sont intrinsèquement plus urgents :

  • Critique : Sécurité récolte, Maladie grave
  • Élevée : Dégât météorologique, Rendement anormal
  • Moyenne : Problèmes de qualité
  • Faible : Alertes informatives
🌱

Phase de culture

L'urgence varie selon la phase de culture :

  • Récolte : Urgence maximale, impact immédiat
  • Pré-récolte : Urgence élevée
  • Verdaison/Nouaison : Urgence moyenne
  • Floraison : Impact sur future récolte
📝

Contenu de la description

Mots-clés dans votre description peuvent augmenter l'urgence :

  • Termes critiques (mort, perte totale, catastrophe)
  • Propagation rapide (épidémie, invasion)
  • Menace immédiate
📊

Historique du verger

Le système considère :

  • Problèmes similaires passés et leur gravité
  • Tendances saisonnières
  • Performance globale du verger

ℹ️ Important : Décrivez vos problèmes en détail pour une meilleure évaluation de l'urgence
